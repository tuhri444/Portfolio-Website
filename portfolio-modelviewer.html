Model Viewer



Description

Throughout my game development career, I have and will continue to stumbled upon opportunities that allow me to learn something new. Including learning how to make 3D art work. This is a little tool I created to display them.

This tool is quite lacking compared to other 3D model viewers that are readily available on the internet. The reason for creating it is simply because it is more fun to try and create my own tool.
